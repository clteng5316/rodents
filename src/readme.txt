開発者向けドキュメント

■ ビルドに必要なもの

無料のツールの組み合わせで、ビルド環境を構築できます。

・Visual C++ 2005 or 2008
	VC++の方言とテンプレートの部分特殊化を使用しています。
・Windows SDK 1.0
	Vista 相当のヘッダとライブラリを使用します。

■ ソースコードの構成

- core
	アプリケーションの実体です。Python モジュールになっています。
- core/xp
	Windows XP 上で、Vista との互換性を実現します。
- graphics.gdiplus
	画像処理モジュールです。GDI+ を利用する実装です。
- include
	共通のヘッダファイルです。
- main
	実際のエントリポイントです。
